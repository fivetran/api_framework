- **⌘ + W** - Close window/tab
- **⌘ + Q** - Quit application
- **⌘ + Space** - Open Spotlight search
- **⌃ + ↑** - Mission Control (all windows)
- **⌃ + ↓** - Application windows
- **F11** - Show desktop

#### Text Editing
- **⌘ + C** - Copy
- **⌘ + V** - Paste
- **⌘ + X** - Cut
- **⌘ + Z** - Undo
- **⌘ + Shift + Z** - Redo
- **⌘ + A** - Select all
- **⌘ + F** - Find

- **⌘ + Shift + G** - Find previous
- **⌘ + B** - Bold
- **⌘ + I** - Italic

## File Management

### Finder Basics

**Finder** is macOS's file manager (equivalent to Windows File Explorer).

#### Key Differences:
- **No drive letters** - Files organized by location, not drive
- **Home folder** - Your personal files are in `/Users/[YourName]`
- **Applications folder** - Apps stored in `/Applications` (not Program Files)
- **Trash** - Deleted files go to Trash (not Recycle Bin)

#### Common Locations:
- **Desktop** - `/Users/[YourName]/Desktop`
- **Documents** - `/Users/[YourName]/Documents`
- **Downloads** - `/Users/[YourName]/Downloads`
- **Applications** - `/Applications`
- **System files** - `/System` (hidden by default)

#### Finder Views:
- **Icon View** (⌘ + 1) - Thumbnail grid
- **List View** (⌘ + 2) - Detailed list with columns
- **Column View** (⌘ + 3) - Hierarchical columns
- **Gallery View** (⌘ + 4) - Large preview with thumbnails

#### Quick Actions:
- **Right-click** (or Control + Click) - Context menu
- **⌘ + I** - Get Info (Properties)
- **⌘ + D** - Duplicate
- **⌘ + Delete** - Move to Trash
- **⌘ + E** - Eject

## Application Management

### Installing Applications

**Windows:** `.exe` installers → **Mac:** `.dmg` files or App Store

1. **App Store** - Similar to Microsoft Store
2. **DMG files** - Download → Open → Drag app to Applications folder
3. **PKG installers** - Double-click and follow wizard
4. **Direct .app files** - Drag to Applications folder

### Running Applications

- **Launchpad** (F4) - iOS-like app launcher
- **Spotlight** (⌘ + Space) - Type app name and press Enter
- **Dock** - Click app icon
- **Finder** - Navigate to Applications folder

### Quitting vs Closing

- **⌘ + W** - Closes window (app may still run)
- **⌘ + Q** - Quits application completely
- **Red dot** (top-left) - Closes window
- **Yellow dot** - Minimizes to Dock
- **Green dot** - Full screen/tile window

### Force Quit

- **⌘ + ⌥ + Esc** - Opens Force Quit dialog
- Select app → Force Quit
- Or: **⌘ + ⌥ + Shift + Esc** - Force quit frontmost app

## System Navigation

### Menu Bar

**Top of screen** (not in each window like Windows)
- **Left side** - Apple menu, app menus
- **Right side** - Status icons (Wi-Fi, battery, time, etc.)

### Dock

**Bottom of screen** (default)
- **Left side** - Application shortcuts
- **Right side** - Folders, Trash
- **Dots** - Indicate running applications
- **Right-click** - Options menu

**Dock Customization:**
- System Settings → Desktop & Dock
- Adjust size, position, auto-hide

### Mission Control

**⌃ + ↑** or **Swipe up with 3-4 fingers**
- View all open windows
- Switch between desktops/spaces
- Drag windows between spaces

### Spotlight Search

**⌘ + Space**
- Search files, apps, emails, contacts
- Calculator, unit conversions
- Web searches
- System preferences

### System Settings

**Apple menu → System Settings** (or **⌘ + ,** in Finder)
- Similar to Windows Control Panel
- Organized by category
- Search bar at top

## Productivity Tips

### Trackpad Gestures

- **Two-finger scroll** - Scroll up/down
- **Two-finger tap** - Right-click
- **Pinch to zoom** - Zoom in/out
- **Swipe left/right** - Navigate back/forward
- **Swipe up** - Mission Control
- **Swipe down** - Show desktop
- **Three-finger drag** - Move windows (enable in Accessibility)

### Multiple Desktops (Spaces)

- **⌃ + ←/→** - Switch between desktops
- **Mission Control** - Create new desktop (drag window to top)
- **Swipe left/right** - Navigate spaces

### Quick Look

- **Select file + Space** - Preview without opening
- Works with documents, images, videos, PDFs
- **⌥ + Space** - Full-screen Quick Look

### Text Expansion

- System Settings → Keyboard → Text Replacements
- Create shortcuts (e.g., "omw" → "On my way!")

### Clipboard History

- **⌘ + Shift + V** - Some apps support clipboard history
- Consider third-party apps like **Paste** or **Clipboard Manager**

### Terminal

- **⌘ + Space** → Type "Terminal"
- Similar to Command Prompt/PowerShell
- Default shell: **zsh** (or bash on older macOS)

### File Paths

- **⌘ + ⌥ + C** - Copy file path in Finder
- **⌘ + Shift + G** - Go to folder (enter path)

### Hidden Files

- **⌘ + Shift + .** - Show/hide hidden files in Finder
- Hidden files start with a dot (`.filename`)

### Emoji & Special Characters

- **⌃ + ⌘ + Space** - Character viewer/emoji picker
- Available in most text fields

## Common Issues & Solutions

### "Application can't be opened"

**Solution:** System Settings → Privacy & Security → Allow app

### Can't find a file

**Solution:** Use Spotlight (⌘ + Space) or Finder search (⌘ + F)

### Application won't quit

**Solution:** ⌘ + ⌥ + Esc → Force Quit

### Need to right-click

**Solution:** Control + Click, or two-finger tap on trackpad

### Can't delete a file

**Solution:** Empty Trash first, or check file permissions (⌘ + I)
